# 示意图名称与图意说明
更新日期：2026-03-25

## 1. 示意图1_系统总体架构.svg
- 图名：新版智能采掘设计系统总体架构示意图
- 图意：展示新版系统从前端交互层、后端服务层到规划与分析模块层的总体分层关系，适合放在“系统架构”章节中。

## 2. 示意图2_数据流转关系.svg
- 图名：多源数据到规划结果的数据流转示意图
- 图意：突出采区边界、钻孔、分层、项目快照等数据如何进入标准化、建模、规划和输出流程，适合放在“数据组织与处理”章节中。

## 3. 示意图3_规划算法协同关系.svg
- 图名：四模式智能规划算法协同关系示意图
- 图意：说明工程效率、资源回收、覆岩扰动和加权综合四类规划模式之间的关系，以及它们如何在候选池、评分和排序中协同。

## 4. 示意图4_设计评价闭环.svg
- 图名：规划设计 - 接续 - 经济评价闭环示意图
- 图意：强调工作面布局方案不是终点，而是进一步进入接续分析、风险联动和经济分析，最终形成闭环评价。

## 5. 示意图5_新旧系统版本边界.svg
- 图名：新旧规划系统版本边界示意图
- 图意：专门区分新版 `GeoMind NEXUS` 平台与旧版 Node 原型系统的关系，用于论文中“系统演进与版本边界”小节，避免把旧版功能误写成新版主线。
