# 论文修改方案

> 目标期刊：《煤炭科学技术》
> 论文题目：采区智能规划设计一体化方法与系统
> 制定日期：2026-04-01

---

## 一、现有初稿问题诊断

### 1.1 参考文献问题（最关键）

| 问题 | 现状 | 期刊要求 | 差距 |
|------|------|----------|------|
| 文献总数 | 20篇 | ≥20篇 | 刚达标，建议增至25-30篇 |
| 近5年占比 | ~10%（仅2篇2021+） | >70% | **严重不足** |
| 期刊文献占比 | ~85% | >80% | 达标 |
| 英文SCI文献 | 仅2篇 | 建议占30-50% | **严重不足** |
| 引用方式 | 笼统引用（如[1,2,3,4,5]） | 需具体论述 | 需改进 |

**核心动作**：用新准备的40篇文献（16篇SCI Q1 + 10篇SCI Q2 + 10篇中文卓越）全面替换现有20篇老文献。

### 1.2 摘要问题

| 检查项 | 现状 | 要求 | 判断 |
|--------|------|------|------|
| 字数 | ~500字 | 500-1000字 | 刚达标，建议扩充至600-800字 |
| 结构 | 背景+方法+结果混合 | 背景、目的、方法和过程、结果、结论 | 需重组 |
| 自评语言 | 有"样例级可行性"等 | 禁止自评 | 需删除 |
| "本文"用词 | 多处出现 | 禁止使用 | 需替换 |
| 分段 | 未分段 | 不分段 | 达标 |

### 1.3 引言问题

期刊要求引言包含3部分：
1. 研究背景
2. 综述与研究内容相关的国内外研究现状及存在问题（**不可笼统引用而不论述**）
3. 引出研究目的和价值

现状：现有引言有框架但综述部分薄弱，引用[1,2,3,4,5]等连号引用缺乏具体论述。

### 1.4 图表问题

| 问题 | 说明 |
|------|------|
| 图题格式 | 需中英文对照（如"图1 采区边界与钻孔分布 Fig.1 Mining area boundary and borehole distribution"） |
| 图件数量 | 现有4张系统生成图 + 5张示意图，建议增至8-12张 |
| 表格 | 当前无表格，建议补充1-2个对比表 |
| 公式 | 需使用Mathtype格式，当前为Markdown格式 |

### 1.5 其他格式问题

- 题目字数：现标题16字，满足<22字要求
- 首页脚注：收稿日期、基金项目、作者简介待填写
- 关键词：现7个，满足5-8个要求
- 中图分类号：待填写

## 二、修改计划（按优先级排序）

### 阶段1：参考文献全面替换（优先级：最高）

**目标**：将现有20篇旧文献替换为新准备的40篇高质量文献，确保近5年>70%、英文SCI占30-50%。

**步骤**：

1. **建立新参考文献编号表**

   按论文正文中首次引用顺序重新编号，拟定分配如下：

   |----------|-------------------|------|
   | 引言-研究背景 | [1]-[5] | 煤矿智能化综述、采区规划需求 |
   | 引言-覆岩与采动损害 | [6]-[12] | 覆岩破坏、采动损害、裂隙演化 |
   | 引言-地表沉陷 | [13]-[15] | 沉陷预测方法 |
   | 引言-采区规划与优化 | [16]-[20] | 采区布局、多目标优化、NSGA-II |
   | 引言-数字孪生与智能系统 | [21]-[24] | 数字孪生、智能矿山 |
   | 方法-数据与场景 | [25]-[27] | 数据管理、场景化分析 |
   | 方法-ODI与覆岩 | [28]-[30] | ODI扰动评价 |
   | 方法-规划算法 | [31]-[33] | 智能规划算法、候选生成 |

   | 方法-经济评价 | [37]-[39] | 工程经济、NPV分析 |
   | 讨论 | [40] | 对比分析补充 |

   旧文献 → 新文献替换建议：

   - 旧[1] 王国法2018智慧煤矿 → 新[35] 煤矿智能化建设高质量发展难题与路径（2024）

3. **补充文献**（如有需要，从参考文献清单中补充）

### 阶段2：摘要重写

**目标**：按照期刊要求重写摘要，满足500-1000字、结构化、无自评语言。

**新摘要结构**：

```
[背景] 采区规划设计在煤矿生产技术决策链中的地位和传统模式的问题（~100字）
[目的] 提出一种采区智能规划设计一体化方法与系统，解决传统割裂式工作模式（~80字）
[方法和过程]
  - 多源数据标准化与项目快照机制（~60字）
  - 三场景分析与地质参数场构建（~60字）
  - ODI风险表征与四模式智能规划（~80字）
  - 三阶段采掘接续与工程经济分析（~60字）
[结果] 系统样例验证结果——工作面数量、巷道数量、关键图件输出（~80字）
[结论] 平台化集成的意义和适用边界（~80字）
```

**注意事项**：
- 不使用"本文"、"作者"、"笔者"
- 不使用自评性语言（如"首次"、"创新性地"）
- 以文字叙述为主，不出现图表和公式
- 英文摘要需对应翻译

### 阶段3：引言文献综述扩写

**目标**：将引言从现有~500字扩展至1500-2000字，形成完整的文献综述。

**引言结构**（编号从0开始）：

```
0 引言

0.1 研究背景（~300字）
  - 采区规划在煤矿生产中的定位
  - 传统规划模式的问题
  - 智能化矿山建设的政策和技术背景
  引用：[1]-[5]

0.2 国内外研究现状（~1200字）
  分4个方向综述：

  (1) 覆岩破坏与采动损害研究（~300字）
    - 覆岩裂隙演化规律
    - 采动应力扰动分析
    - 导水裂隙带与含水层保护
    引用：[6]-[15]
    存在问题：多为单一场景研究，缺乏多场景统一评价框架

  (2) 采区规划与工作面布置优化（~300字）
    - 地下采场布局优化方法
    - 多目标优化算法（NSGA-II等）在矿山中的应用
    - 生产调度与接续优化
    引用：[16]-[23]
    存在问题：布局优化与风险评价、经济评价割裂

  (3) 数字孪生与地质建模（~300字）
    - 矿山数字孪生技术
    - 地质参数场构建方法
    - 多源数据融合建模
    引用：[24]-[30]
    存在问题：建模结果与规划决策之间的传递链路不完整

  (4) 工程经济评价（~300字）
    - 矿山经济评价方法
    - 不确定性条件下的投资决策
    - 现金流模型与NPV分析
    引用：[37]-[39]
    存在问题：经济评价通常脱离规划主流程

0.3 现存问题与本文工作（~400字）
  - 总结上述4个方向的共同问题：割裂式工作、缺乏统一数据底座
  - 引出本文工作：采区智能规划设计一体化方法与系统
  - 说明主要贡献：4点
```
