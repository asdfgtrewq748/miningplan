# 协同调控：ODI 统一标尺口径（确认版）

日期：2026-01-26

## 目标
在“原方案 ODI”与“协同调控方案 ODI”（以及不同阶段/不同工作面序号对应的 ODI）之间建立**同一归一化标尺**，最终只输出**一张 ODI 空间分布图**（同色标、可比较）。

## 口径结论（已确认）
- 联合样本集 **包含“地质-only”点**：用于稳定全局标尺，提升跨方案可比性。
- 统一归一化采用 **联合分位数 P5/P95**。
- 推进时序不引入“推进速度”，以**离散工作面序号**表达推进顺序；“动态更新”用**用户可配置段数 N**（分段/阶段）表达。

## 定义与符号
- $S_\text{geo}$：地质-only 评价点集合（geo-only 参数提取链路得到）。
- $S_\text{plan}$：方案相关评价点集合（full/high-precision 链路得到；随工作面几何/采高等变化）。
- 联合样本集：
  $$S = S_\text{geo} \cup S_\text{plan}$$
- $ODI_\text{raw}(p)$：点 $p$ 的 ODI 原始聚合值（保持“本次 computeOdi 的 raw 聚合定义”，不在此文改变公式）。
- $q_5, q_{95}$：对集合 $\{ODI_\text{raw}(p)\mid p\in S\}$ 计算的 5% 与 95% 分位数。

## 统一标尺归一化（norm*）
对任意点 $p$：

1) 分位数截断（winsorize）：
$$ODI_\text{clip}(p)=\min(\max(ODI_\text{raw}(p), q_5), q_{95})$$

2) 线性映射到 $[0,1]$：
$$ODI^*(p)=\frac{ODI_\text{clip}(p)-q_5}{q_{95}-q_5}$$

3) 数值边界：
- 若 $q_{95}=q_5$（极端退化），则定义 $ODI^*(p)=0.5$ 或回退到 min-max（需实现时二选一，推荐 0.5 以避免噪声放大）。

> 说明：$ODI^*$ 即“统一标尺 ODI（norm*）”。最终插值成场、扰动采样评分、热力图色标都以 $ODI^*$ 为准；$ODI_\text{raw}$ 仅用于调试/追溯。

## 阶段/分段（N 段）
### 输入
- $N$：用户可配置正整数，表示推进阶段数。
- 工作面推进顺序：离散序号（例如 1,2,3,...）。

### 建议默认
- 默认 $N=5$（可在 UI 中暴露并允许修改）。

### 语义
- 第 $k$ 段（$k\in[1,N]$）代表推进离散过程中的一个“快照”。
- 每段对应一套 $S_\text{plan}^{(k)}$ 与 $ODI_\text{raw}^{(k)}$；但 **归一化标尺共享同一对 $q_5/q_{95}$**（由联合样本集 $S$ 计算），从而跨段可比较。

## 联合样本集的构造建议
为保持标尺稳定，推荐：
- $S_\text{geo}$ 始终纳入（地质-only 点）。
- $S_\text{plan}$ 取“原方案 + 当前协同方案”的评价点并集；若存在阶段/分段，取所有段的并集（或至少取原方案与当前段并集，取决于性能预算）。

## 输出（本阶段只要一张图）
- 输出 ODI 场：以 $ODI^*$ 插值成单一栅格场（同色标，范围固定 $[0,1]$）。
- 可选的辅输出（不强制）：
  - $q_5, q_{95}$ 用于可解释性与版本复现；
  - 标尺样本量 |S| 与来源统计（geo-only/plan 各自数量）。

## 需要后续实现时确认的细节（不改变口径）
- $q_{95}=q_5$ 的回退策略（0.5 vs min-max）。
- 分位数计算方法：
  - 对离散样本采用线性插值分位数（常见实现）或 nearest-rank；保持前后端一致。
- 性能：若 S 很大，分位数可用采样或 streaming quantile（例如 t-digest）近似。
